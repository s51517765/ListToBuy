# ListToBuy

## 概要 (Overview)

FirebaseのリアルタイムデータベースとStorageを利用しています。

主な機能は以下です。

・欲しいものを追加する

・欲しいものに写真を添付する

・入手済みリストを保持する

・スマホ（Android / iOS）から利用可能 


## 動作要件 (Requirements)

Javascript / Firebase
